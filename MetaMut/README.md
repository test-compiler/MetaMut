This project is our implementation of MetaMut.

**Requirements**:
- `watchdog`
- `numpy`
- `psutil`
- `sysv_ipc`
- `python` (>= 3.9.0)

**Run**:
To run MetaMut, you'll first need to config azure openai keys in [configs.py](scripts/configs.py).

Then, follow these steps to run:
```
python3 scripts/main.py
```
